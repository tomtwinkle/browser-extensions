# browser-extensions

Chrome / Edge 拡張機能のモノレポです。

[English README](README.md)

## Extensions

| 拡張機能 | 概要 | README |
|---|---|---|
| [exif-viewer](exif-viewer/) | Hover EXIF Viewer – 画像ホバー時のみ EXIF ボタンを表示し、EXIF 情報をページ内で確認 | [README](exif-viewer/README.ja.md) |
| [meet-translator](meet-translator/) | Google Meet 自動翻訳チャット – 音声をリアルタイムで文字起こし・翻訳し Meet チャットに投稿 | [README](meet-translator/README.ja.md) |
